# Nuvola Programming Language

A completely custom-built programming language designed to handle anything and everything within one unified program.

**Nuvola** is an indentation-based, dynamically typed language with powerful features including concurrency, pattern matching, lambdas, built-in HTTP server, JSON support, regex, file I/O, and direct C interop.

It includes a self-hosted compiler that emits C code, compiled to native binaries.
